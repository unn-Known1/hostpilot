## Features

- 🎨 Beautiful modern UI with dark/light mode
- ⚡ Easy add, edit, delete host entries
- 🔒 Enable/disable entries without deleting
- 🔍 Search and filter functionality
- 💾 Backup & restore feature
- 📥 Import & export hosts files
- 🌐 Cross-platform (Windows, macOS, Linux)

## Quick Start

```bash
# Install dependencies
npm install

# Run in development
npm run dev

# Build for production
npm run build
```

## Usage

### Download Releases
- **Windows**: HostPilot.exe
- **macOS**: HostPilot.dmg
- **Linux**: HostPilot.AppImage

- Electron + React + TailwindCSS + Vite
